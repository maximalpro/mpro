Технологии которыми я использую

Этот раздел, больше напоминает мое резюме, и даст Вам возможность понять мою квалификацию, даже если Вы не обладаете большими познаниями в IT.

Начну с того, что за годы работы я хорошо освоил достаточно большое количество разных технологий, которые были связаны как напрямую с веб разработкой, так и не связаны вовсе.

Это помогло мне научиться создавать не просто «сайтики», но и сложные серьёзные проекты.


Вот краткий список того, чем я пользуюсь последние пару лет для разработки:

Языки программирования:

PHP и С++ для back end разработки

JavaScript (jQuery, JSON, AJAX), HTML (XHTML, HTML5), XML, CSS3 для front-end

Базы данных:

MySQL, PostgreSQL, DynamoDB

Вэб-сереверы:

Apache, nginx, LightTPD

в основном использую связку Apache+nginx

Системы ускорения работы веб-приложений:

eAccelerator для ускорения работы PHP

memcached для ускорения работы БД

Основные браузеры для которых я разрабатываю и оптимизирую веб-приложения:

Chrome, Firefox, Opera, Internet Explorer, Safari


Большинство проектов для повышения безопасности и экономии средств заказчиков я перевожу на Amazon Web Services, где полностью настраиваю сервер EC2 на ОС Ubuntu и настраиваю отдельный сервер БД.

Для более требовательных заказчиков я использую больший набор сервисов от AWS таких как LoadBalancer, Route53 и т.д.

В программировании я стараюсь не использовать Фреймворки для back-end разработки, чтобы минимизировать нагрузку на сервер, ведь только один Zend может подключаться дольше всего сценария, если его не объединить в один файл и не кэшировать в байт коде eAccelerator-ом.
